# 🎮 Bedrock CodeBuilder 사용 가이드

마인크래프트 베드락 에디션에서 에이전트를 블록 코딩으로 제어할 수 있는 프로그램입니다.

## 📋 시스템 요구사항

- **Windows 10/11** (64비트)
- **마인크래프트 베드락 에디션** (Windows 10 Edition)
- **인터넷 브라우저** (Chrome, Edge 등)

## 🚀 설치 및 실행

### 1단계: 초기 설정
1. `setup.bat` 파일을 **마우스 우클릭**
2. **"관리자 권한으로 실행"** 선택
3. "설정이 완료되었습니다!" 메시지 확인

### 2단계: 프로그램 실행
1. `Bedrock-CodeBuilder.exe` 더블클릭으로 실행
2. 잠시 후 브라우저에서 관리 페이지가 자동으로 열림
3. "✅ 코드빌더 실행가능" 메시지 확인

### 3단계: 마인크래프트 연결
1. **마인크래프트 베드락 에디션** 실행
2. 월드에 입장 (크리에이티브 모드 권장)
3. 채팅창 열기 (**T키**)
4. 관리 페이지에 표시된 연결 명령어 입력
   ```
   /connect localhost:3000
   ```
5. 엔터키를 누르면 자동으로 블록 코딩 페이지가 열림

## 🧩 블록 코딩 사용법

### 기본 사용법
1. **Hat 블록**: 명령어나 이벤트로 코드 실행 시작
2. **Agent 블록**: 에이전트 움직임 및 동작 제어
3. **실행**: 마인크래프트에서 설정한 명령어 입력

### 주요 블록 종류
- **🎩 Hat 블록**: 채팅 명령어, 아이템 획득, 블록 설치/파괴 이벤트
- **🤖 Agent 블록**: 이동, 회전, 블록 설치/파괴, 텔레포트
- **📍 좌표 블록**: 절대좌표, 상대좌표, 카메라 상대좌표
- **🧱 블록 타입**: 다양한 마인크래프트 블록 종류

### 실행 예시
1. "안녕" 명령어로 에이전트가 앞으로 5칸 이동하는 코드:
   - Hat 블록: "채팅 명령어 받았을 때: 안녕"
   - Agent 블록: "앞으로 이동" x5개 연결
2. 마인크래프트 채팅창에 "안녕" 입력
3. 에이전트가 앞으로 5칸 이동

## 🔧 프로그램 종료

- **정상 종료**: 브라우저의 관리 페이지 창을 닫으면 프로그램이 종료됩니다
- **강제 종료**: 작업 관리자에서 Bedrock-CodeBuilder.exe 프로세스 종료

## ❓ 문제 해결

### 연결이 안 될 때
1. `setup.bat`를 다시 관리자 권한으로 실행
2. 마인크래프트 베드락 에디션인지 확인 (Java Edition 불가)
3. 방화벽에서 프로그램 허용 확인

### 명령어가 작동하지 않을 때
1. 월드가 **크리에이티브 모드**인지 확인
2. **명령어 실행 권한**이 있는지 확인
3. 에이전트가 소환되어 있는지 확인 (자동으로 소환됨)

### 프로그램이 실행되지 않을 때
1. Windows Defender나 백신 프로그램에서 차단하는지 확인
2. 다른 프로그램이 포트 3000, 4000을 사용하고 있는지 확인
3. 관리자 권한으로 프로그램 실행 시도

## 💡 사용 팁

- **저장/불러오기**: 만든 블록 코드는 저장하고 불러올 수 있습니다
- **여러 명령어**: 한 번에 여러 명령어를 등록할 수 있습니다  
- **이벤트 활용**: 아이템 획득, 블록 설치 등의 이벤트로 자동 실행 가능
- **좌표계 이해**: 절대좌표, 상대좌표, 카메라 상대좌표를 잘 활용하세요
